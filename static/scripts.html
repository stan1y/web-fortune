Web fortune teller is developer friendly! You can use it to power your websites and applications with everyday wisdom absolutely for free. Getting a wisdom programmatically is very simple. All you need to do is to issue a GET request to this url. Here is a list of arguments you can use to tailor the result for your needs. All of them are optional:
  • owner - specify a person who uploaded a quote, it's nickname in google's profile. It can be you or anybody else. Default: everybody
  • format - in which format you want to have a response. Currently these formats are supported:
    • html - an html page, the default one
    • json - a json encoded dictionary
    • jsonp - a json with padding for cross-domain calls. Requires a callback argument
  • callback or jsonp - a callback function used to parse json on your page.
